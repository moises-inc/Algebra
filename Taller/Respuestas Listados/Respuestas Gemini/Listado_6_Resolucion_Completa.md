## 1. Conversión de Unidades (Grados a Radianes y viceversa)
*Utilizando la relación $\pi \text{ rad} = 180^\circ$.*

- a) $20^\circ$ a radianes: $20^\circ \cdot \frac{\pi}{180^\circ} = \frac{\pi}{9} \text{ rad}$.
- b) $\frac{7\pi}{6} \text{ rad}$ a grados: $\frac{7\pi}{6} \cdot \frac{180^\circ}{\pi} = 7 \cdot 30^\circ = 210^\circ$.
- c) $15^\circ$ a radianes: $15^\circ \cdot \frac{\pi}{180^\circ} = \frac{\pi}{12} \text{ rad}$.
- d) $\frac{2\pi}{5} \text{ rad}$ a grados: $\frac{2\pi}{5} \cdot \frac{180^\circ}{\pi} = 2 \cdot 36^\circ = 72^\circ$.
- e) $-120^\circ$ a radianes: $-120^\circ \cdot \frac{\pi}{180^\circ} = -\frac{2\pi}{3} \text{ rad}$.
- f) $-250^\circ$ a radianes: $-250^\circ \cdot \frac{\pi}{180^\circ} = -\frac{25\pi}{18} \text{ rad}$.
- g) $-90^\circ$ a radianes: $-90^\circ \cdot \frac{\pi}{180^\circ} = -\frac{\pi}{2} \text{ rad}$.

## 2. Determinación de ángulos $\theta$ en radianes
- a) $\sin \theta = 0.8 \implies \theta = \arcsin(0.8) \approx 0.93 \text{ rad}$.
- b) $\tan \theta = 1 \implies \theta = \frac{\pi}{4} \text{ rad}$ (en el primer cuadrante).
- c) $\sin \theta = 1/2 \implies \theta = \frac{\pi}{6}, \frac{5\pi}{6} \text{ rad}$.
- d) $\sin \theta = \frac{\sqrt{2}}{2} \implies \theta = \frac{\pi}{4}, \frac{3\pi}{4} \text{ rad}$.
- e) $\tan \theta = \sqrt{3} \implies \theta = \frac{\pi}{3}, \frac{4\pi}{3} \text{ rad}$.
- f) $\cos \theta = 1/2 \implies \theta = \frac{\pi}{3}, \frac{5\pi}{3} \text{ rad}$.

## 3. Resolución de Triángulos Rectángulos (Identidades Básicas)
- a) Dado $\text{hipotenusa} = 18$ y ángulo $60^\circ$: $y = 18 \cdot \sin(60^\circ) = 18 \cdot \frac{\sqrt{3}}{2} = 9\sqrt{3}$.
- b) Dado $\text{cateto opuesto} = 30$ y ángulo $30^\circ$: $x = \frac{30}{\tan(30^\circ)} = \frac{30}{1/\sqrt{3}} = 30\sqrt{3}$.
- c) Dado $\text{cateto opuesto} = 32$ y ángulo $60^\circ$: $x = \frac{32}{\tan(60^\circ)} = \frac{32}{\sqrt{3}} = \frac{32\sqrt{3}}{3} \approx 18.5$.
- d) Dado $\text{cateto opuesto} = 20$ y ángulo $45^\circ$: $r = \frac{20}{\sin(45^\circ)} = \frac{20}{\sqrt{2}/2} = \frac{40}{\sqrt{2}} = 20\sqrt{2}$.

## 18. Simplificación de Expresiones
- a) $\frac{1}{1+\cos x} + \frac{1}{1-\cos x} = \frac{(1-\cos x) + (1+\cos x)}{1-\cos^2 x} = \frac{2}{\sin^2 x} = 2\csc^2 x$.
- b) $\frac{\cos x}{1+\sin x} + \frac{1+\sin x}{\cos x} = \frac{\cos^2 x + (1+\sin x)^2}{\cos x(1+\sin x)} = \frac{\cos^2 x + 1 + 2\sin x + \sin^2 x}{\cos x(1+\sin x)} = \frac{2 + 2\sin x}{\cos x(1+\sin x)} = \frac{2(1+\sin x)}{\cos x(1+\sin x)} = 2\sec x$.
- c) $\tan x + \frac{\cos x}{1+\sin x} = \frac{\sin x}{\cos x} + \frac{\cos x}{1+\sin x} = \frac{\sin x + \sin^2 x + \cos^2 x}{\cos x(1+\sin x)} = \frac{\sin x + 1}{\cos x(1+\sin x)} = \frac{1}{\cos x} = \sec x$.

## 20. Ecuaciones Trigonométricas en $[0, 2\pi[$
- a) $-\csc x = 1 \implies \sin x = -1 \implies x = \frac{3\pi}{2}$.
- e) $2\cos^2 x - 3\cos x - 2 = 0$. Sea $u = \cos x$: $2u^2 - 3u - 2 = 0 \implies (2u + 1)(u - 2) = 0$.
   - $\cos x = -1/2 \implies x = \frac{2\pi}{3}, \frac{4\pi}{3}$.
   - $\cos x = 2$ (Sin solución en $\mathbb{R}$).
- h) $\sin 2x + \sin x = 0 \implies 2\sin x \cos x + \sin x = 0 \implies \sin x (2\cos x + 1) = 0$.
   - $\sin x = 0 \implies x = 0, \pi$.
   - $\cos x = -1/2 \implies x = \frac{2\pi}{3}, \frac{4\pi}{3}$.
